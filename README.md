# Student Management System

A Java Swing-based Student Management System with Login Authentication, Dashboard, Student Management, Docker support, Jenkins CI/CD integration, and SonarQube configuration.

---

## Features

- User Login Authentication
- Dashboard UI
- Add Student
- Delete Student
- Search Students in Real-Time
- JTable-based Student Records
- Duplicate Roll Number Validation
- Clean Layered Architecture
- Docker Support
- Jenkins CI/CD Pipeline
- SonarQube Integration

---

## Technologies Used

- Java
- Java Swing
- Git & GitHub
- Docker
- Jenkins
- SonarQube
